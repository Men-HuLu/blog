#### 常用传输协议
- 哑协议
- 智能协议 file:///
- http/https
- ssh协议


```
pwd                         //查看路径
git clone 路径              //哑协议路径
git clone file:///路径      //智能协议路径
```


### 创建SSH公私钥

```
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```
