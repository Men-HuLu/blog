### 工作区到暂存区
![image](https://git-scm.com/book/en/v2/images/lifecycle.png)

你工作目录下的每一个文件都不外乎这两种状态：已跟踪或未跟踪。
从工作区到暂存区无疑有4种状态
- 未跟踪
- 未修改
- 已修改
- 暂存区



### 暂存区提交到版本库
![image](https://git-scm.com/book/en/v2/images/commit-and-tree.png)

#### 如图所示:
##### Git 仓库中主要有3类对象
- commit:提交时间的快照,包含1个树和提交信息
- tree：树里边可以包含树和二进制,图中只包含文件
- blob：二进制文件
