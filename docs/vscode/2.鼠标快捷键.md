##### 1.文本选择
```
双击左键两次选单词
三次左键这⼀⾏代码
四次左键整个⽂档

单击行号,选一行
上下移动,就能选择多行
```

##### 2.⽂本编辑
```
点击选择块代码拖动
鼠标移到单词上,按ctrl，可以看到函数的实现
Ctrl + ⿏标左键(超链接)
```

##### 3.多光标
```
按鼠标中间
```

##### 4.快速预览 
```
Ctrl+空格键
```

##### 5.参数预览
```
Ctrl+Shift+Space        //参数预览
```

##### 6.快速修复
```
点击⻩⾊的灯泡图标
Ctrl + .
```

##### 7.重构
```
函数或变量上按F2,出现的地⽅就都会被修改.
```
