## 新建仓库git init

##### 建git仓库的两种场景：
1.把已有项目代码纳入git管理
```
cd 项目代码所在文件夹
git init
```
2.新建的项目直接用Git管理

```
git init your_project
cd your_project
```

## 克隆仓库git clone

```
默认使用版本库文件名
 git clone https://github.com/Men-HuLu/git_learning.git 
 
使用自定义名称
 git clone https://github.com/Men-HuLu/git_learning.git learing
```
