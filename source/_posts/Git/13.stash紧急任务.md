### 1.先保存到贮藏区
```
git add -u  //保存
git stash   //存放到贮藏区
```
### 2.贮藏区查看

```
git stash list
```
### 3.从贮藏区拿出
```
git stash apply             //不删除stash
git stash apply stash@{1}   //取出第二个(索引为1),不删除stash
git stash pop               //删除stash
```

### 3.从贮藏区删除
```
git stash clear                 //贮藏区全部清空
git stash drop stash@{$num}     //丢弃stash@{$num}存储，从列表中删除这个存储       
```
